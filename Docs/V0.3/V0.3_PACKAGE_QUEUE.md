UI Runtime Lock Remediation / 沉余 fallback 分包收口：

```text
GUARD_SYNC_UI_RUNTIME_LOCK_REMEDIATION_PLAN_ACCEPTED

问题判断：
- BottomNavBar_Root 等 UI 在 Play 后回退，不是单点 Snapshot 工具失败。

- Builder 期允许生成骨架。
```
