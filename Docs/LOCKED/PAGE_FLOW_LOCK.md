# PAGE FLOW LOCK

## 页面职责

- 首页 = 照灯小铺空间化导航
- 试炼 = 主线自动战斗 / 巡行推进
- 炼符 = 符箓培养 / 成长
- 探索 = 三选一 / 随机节点 / 秘境 / 天机炉方向
- 后屋 = 后续生产养成，不承担试炼同类挂机收益
- 梦签 = 每日签
- 道藏 / 符箓库 = 图鉴 / 收藏 / 卡册，不承担战斗摆放
- 战斗背包 / 阵盘 = 试炼和战斗流程内功能，不是一级导航页签

## 底栏功能方向

二级页应提供用于回到小店、进入试炼、进入炼符、进入探索占位、进入更多页的导航能力。

本文只锁底栏的功能职责和禁止项，不锁普通实现细节。

不在本文中锁定：

- 按钮先后顺序
- 尺寸、颜色、位置、间距
- 临时文案
- 图标表现
- 动效与灰盒布局形式

## 禁止

- 底栏出现“背包”
- 底栏出现 `BattleBackpack`
- 道藏 / 符箓库承担战斗摆放
- 后屋承担试炼挂机收益
- PVP 命名为“论道”

页面职责、底栏功能边界或禁止项需要调整时，至少按黄灯处理；若调整会触碰战斗状态、Root 或 RunFlow，则按红灯处理。

仅调整按钮顺序、视觉布局、颜色、尺寸、间距、临时文案或图标表现，且不改变页面职责与功能边界时，不视为修改本锁定文档。
