## 当前版本交付形态硬边界

- 首页必须以独立的新场景交付，不得用旧 `FormationCounter` 场景中的隐藏面板冒充新首页场景。
- 首页在任务档案规定的真实启动路径中默认 active。
- 如果当前任务要求 Play 首帧可见，则进入 Play 后不得依赖测试、反射、调试按钮或人工 `SetActive` 才能显示。
- Unity 重启或重新打开工程后，首页仍必须能按同一真实路径到达和显示。
- 自动测试不得主动打开首页后再证明首页默认可见。
- 本节锁定功能交付形态，不锁 UI 布局、导航先后顺序、颜色、尺寸、文案或局部代码组织。
